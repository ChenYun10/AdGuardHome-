Usage of the updates.sh script
Taking Debian as an example:
Open the terminal and run:
wget https://github.com/ChenYun10/AdGuardHome-/releases/download/update/updates.sh
nano updates.sh
In the first line, change TARGET_DIR=" to the actual directory where AdGuardHome is located"
Example: TARGET_DIR="/home/shdmt/AdGuardHome"
Then press Ctrl+O to save
And run bash updates.sh
